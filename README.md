# TO-DO APP

A modern Android to-do list application built with Kotlin and Jetpack Compose, designed to help users manage their daily tasks efficiently.

## 📋 Project Overview

TO-DO APP is a feature-rich task management application that provides a clean and intuitive user interface for creating, managing, and tracking to-do items.

## 🛠️ Tech Stack

- **Language**: Kotlin (93.4% of codebase)
- **UI Framework**: Jetpack Compose
- **Target SDK**: Android 15 (SDK 36)
- **Minimum SDK**: Android 7.0 (SDK 24)
- **Build System**: Gradle with Kotlin DSL
